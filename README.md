## ภารกิจพิชิต I (The Conquest of I’s)

เกมการเรียนรู้เชิงสถานการณ์ (Scenario-based Learning) สำหรับนักศึกษาพยาบาลและบุคลากรสุขภาพ เพื่อฝึกทักษะการพยาบาลผู้สูงอายุผ่านกลุ่มอาการสูงอายุ (Geriatric Syndromes) ที่สำคัญ 6 ประการ (6 I’s)

## รายละเอียดโครงการ

เกมนี้ถูกออกแบบมาเพื่อสร้างความตระหนักและการเรียนรู้ในการจัดการปัญหาที่พบบ่อยในผู้สูงอายุ โดยเน้นการวิเคราะห์กรณีศึกษา การตัดสินใจเชิงคลินิก และการประยุกต์ใช้ความรู้ด้านการพยาบาลในสถานการณ์จำลอง

---

## กลุ่มอาการ 6 I's ในเกม

- **Instability:** ความไม่มั่นคงในการทรงตัวและการป้องกันการหกล้ม
- **Immobility:** การจำกัดการเคลื่อนไหวและการจัดการแผลกดทับ
- **Intellectual Impairment:** ภาวะสับสนเฉียบพลัน (Delirium) และสมองเสื่อม
- **Insomnia:** ปัญหาการนอนหลับและสุขอนามัยการนอน
- **Inanition:** ภาวะขาดสารอาหารและปัจจัยเสี่ยงด้านโภชนาการ
- **Incontinence:** ภาวะกลั้นปัสสาวะไม่อยู่และการฝึกกล้ามเนื้ออุ้งเชิงกราน

---

## รูปแบบการเล่น (Gameplay)

เกมแบ่งออกเป็นด่านตามกลุ่มอาการ โดยมีกลไกที่หลากหลาย เช่น:

**Point & Click:** ค้นหาจุดอันตรายในบ้าน (ด่าน Instability) เพื่อจัดสภาพแวดล้อมใหม่

**Visual Puzzle:** เกมจับผิดภาพเพื่อหาปัจจัยเสี่ยงภาวะขาดสารอาหาร (ด่าน Inanition)

**Interactive Scenario:** เลือกชนิดของภาวะกลั้นปัสสาวะและวิธีการพยาบาลที่เหมาะสมจากกรณีศึกษา

**Quiz Mode:** แบบทดสอบปรนัยเพื่อประเมินความรู้ก่อนและหลังเล่น (Pre-Post Test)

---

## ภาพรวมระบบเว็บการเรียนรู้

แพลตฟอร์มประกอบด้วยส่วนสำคัญดังนี้:

- ระบบสมาชิก (Register / Login)
- แบบทดสอบก่อนเรียน (Pre-test)
- Unity WebGL Game เพื่อการเรียนรู้เชิงสถานการณ์
- แบบทดสอบหลังเรียน (Post-test)
- ระบบเกียรติบัตร (Certificate) เมื่อผ่านเกณฑ์
- หน้าครูและแอดมินสำหรับดูรายงานและจัดการข้อมูล

เงื่อนไขการปลดล็อกหลัก:

- ทำ Pre-test ก่อน จึงจะปลดล็อกเกมและ Post-test
- เมื่อ Post-test มากกว่า 60% จะปลดล็อกการรับเกียรติบัตร

## องค์ความรู้ที่สอดแทรก

**การจัดการสภาพแวดล้อม:** การจัดเก็บสายไฟ พรม และการเพิ่มราวจับเพื่อความปลอดภัย

**การพยาบาลเชิงเทคนิค:** การพลิกตะแคงตัวผู้ป่วยติดเตียง และการคัดกรองระดับแผลกดทับ

**การประเมินทางจิตสังคม:** การแยกอาการ Delirium ออกจาก Dementia และการประเมินปัจจัยกระตุ้นความสับสน

---

## วัตถุประสงค์การเรียนรู้

- เพื่อให้ผู้เล่นสามารถระบุปัจจัยเสี่ยงที่ก่อให้เกิดอุบัติเหตุและภาวะแทรกซ้อนในผู้สูงอายุได้

- เพื่อให้ผู้เล่นตัดสินใจเลือกกิจกรรมการพยาบาล (Nursing Intervention) ได้อย่างถูกต้องตามปัญหาของผู้ป่วย

- เพื่อส่งเสริมคุณภาพชีวิตที่ดีและยั่งยืนของผู้สูงอายุ

---

## วิธีเริ่มใช้งานโครงการ (แบบย่อ)

1. ติดตั้ง dependencies

   ```bash
   npm install
   ```

2. เตรียมค่าระบบ Supabase

   ตั้งค่า URL และ Key ในไฟล์ client ให้ถูกต้องตามโปรเจกต์

3. เปิดใช้งานหน้าเว็บจากโฟลเดอร์ public

   สามารถใช้ static server ที่ถนัด เช่น VS Code Live Server

4. ตรวจสอบ Authentication Redirect URLs ใน Supabase

   ควรเพิ่ม URL สำหรับ production และหน้า reset password ให้ครบ
